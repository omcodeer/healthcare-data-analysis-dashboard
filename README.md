# Healthcare Data Analysis Dashboard

## 📌 Project Overview
This project analyzes the care transition pipeline:
CBP Custody → HHS Care → Discharge

## 🎯 Objectives
- Measure system efficiency
- Identify bottlenecks
- Analyze outcomes

## 📊 KPIs
- Transfer Efficiency
- Discharge Effectiveness
- Backlog Analysis

## 🔍 Key Insights
- Low discharge rate is the main bottleneck
- Moderate transfer efficiency
- Backlog indicates system delay
